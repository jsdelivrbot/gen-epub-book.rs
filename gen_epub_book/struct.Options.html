Representation of the application's all configurable values.

Fields

source_file: Option<(String, PathBuf)>

The descriptor file, or None for stdin.

include_directories: Vec<IncludeDirectory>

The roots for relative source paths, or, so called -Include dirs.

Default:

  • ["."] if source_file == None, or
  • [directory containing source_file] otherwise.
output_file: Option<(String, PathBuf)>

The file to insert the assembled ePub to, or None for stdout.

verbose: bool

Whether to print more information.

Default: false

separator: String

Chosen separator.

Default: ":"

free_date: bool

Whether to (attempt to) parse more date formats than just RFC3339.

Default: false
